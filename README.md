# Reinforcement Learning Experiments

This repository includes classic reinforcement learning problems implemented in Python. The goal is to explore fundamental RL concepts through hands-on experiments using clean and minimal code.

## 📂 Contents

- `Bandit.py` – Multi-armed bandit environment using ε-greedy exploration.
- `Blackjack.py` – Monte Carlo control algorithm applied to a simplified Blackjack game.
- `Grid_World.py` – Grid world navigation using value iteration and policy evaluation.

## 🧠 Algorithms Included

- Epsilon-Greedy Action Selection
- Monte Carlo Control (First Visit)
- Policy Evaluation
- Value Iteration

## 🛠️ Technologies

- Python 3.x
- NumPy

## 🚀 How to Run

Each script is standalone and can be run directly:

```bash
python3 Bandit.py
python3 Blackjack.py
python3 Grid_World.py
